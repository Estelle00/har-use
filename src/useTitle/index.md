# useTitle

## 介绍
响应式修改页面标题

## 代码演示

### 基本用法
@import "./demo/index.vue"


## API


### 参数

| 参数       | 说明     | 类型      | 默认值   |
|----------|--------|---------|-------|
| newTitle | 初始化标题  | string \| null \|undefined |null|
| options | 可选的配置项 | [Options](#Options)| {}    |

### Options

| 参数 | 说明             | 类型 | 默认值 |
| --- |-----------------| -- | --- |
| template | 标题模板     | string | %s |
